REMNANT

Jam #1 in Innovation. A time-loop roguelike where every death teaches the world.

SCREENSHOT / TRAILER

OVERVIEW

Remnant Loop was built solo in 48 hours for a themed game jam. The central mechanic: when you die, the world remembers. Enemies adapt to how you fought them. Environmental traps reset but retain memory of which ones you triggered. The dungeon layout subtly shifts.

The result is a roguelike that gets harder in a deeply personal way — it learns you specifically.

MY ROLE

Everything — design, code, pixel art, sound effects. Built solo under a 48-hour time constraint. The adaptive AI and persistence system were the core technical bets I made, and both paid off.

TECHNICAL HIGHLIGHTS

  • Lightweight behaviour tree AI with per-run memory serialisation
  • Enemy adaptation system — tracks player movement patterns, weapon usage frequency, and route preferences across deaths
  • Procedural dungeon generation with biased weighting based on past player paths
  • Entire project built in Godot 4 — first time using the engine seriously
