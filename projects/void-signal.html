VOID
SIGNAL

A minimalist sci-fi puzzle game about decoding alien transmissions.

SCREENSHOT / TRAILER

OVERVIEW

Void Signal places the player at a deep-space monitoring station, intercepting fragmented transmissions from an unknown source. Each level is a signal — a series of encoded patterns the player must analyse, interpret, and decode.

The game features a fully procedurally generated signal system ensuring every playthrough offers fresh challenges, layered against 40+ handcrafted story levels that build the narrative piece by piece.

MY ROLE

Solo developer. Responsible for all game design, programming, procedural generation systems, UI design, and sound design. The visual aesthetic was a deliberate design choice — monochrome with a single accent frequency to mirror the signal-decoding mechanic.

TECHNICAL HIGHLIGHTS

  • Custom procedural signal generator — weighted grammar-based patterns with difficulty scaling
  • Shader-based waveform visualiser written in HLSL
  • Save system with full puzzle state serialisation
  • Accessibility options: colour-blind modes, timing adjustments, hint system
