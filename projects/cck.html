05 VR AUTHORING PLATFORM — INTERNAL TOOL

CONTENT
CREATION KIT

A node-based VR scenario authoring platform — letting non-developers build and deploy interactive VR training without writing code.

OVERVIEW

The Content Creation Kit (CCK) is an in-editor tooling platform built on top of Unity that enables instructional designers and content teams to create, preview, and deploy VR training scenarios without writing code.

It exposes a visual node-based editor for branching dialogue and scenario flow, a categorised asset library of environments and props, and a scene management system for assembling multi-scene training modules — all within a custom Unity Editor UI layer.

MY ROLE

Lead tooling engineer. I designed and built the entire CCK from scratch inside the Unity Editor — including the custom node graph window, the asset library browser, the scene configuration system, and the one-click Quest build pipeline. Used internally to ship 5 of the 9 projects in this portfolio.

TECHNICAL HIGHLIGHTS

  • Custom Unity Editor window — full node graph with pan/zoom, connection routing, and undo/redo stack
  • Serialised scenario format (JSON-based) for version control and team collaboration
  • Asset library system with tagging, search, and drag-to-scene placement
  • Runtime dialogue engine that consumes authored node graphs at play-time with zero additional setup
  • Automated Meta Quest build pipeline — one click to package and push to connected devices
