09 2D GAME — FOCUS & ATTENTION

F★CUS

A pixel-art 2D game blending knowledge quiz questions with distraction management — simulating real cognitive load through play.

OVERVIEW

Focus is a pixel-art 2D game that teaches attention and concentration through a dual-challenge mechanic — players must answer knowledge quiz questions correctly while simultaneously managing on-screen distractions in a dynamic game environment.

As levels progress, environments become noisier and more visually complex, deliberately simulating the conditions under which real cognitive performance degrades. Succeeding requires players to develop genuine attentional control, not just memorise answers.

MY ROLE

Lead developer. I built the distraction detection and scoring system, the progressive difficulty scaling, all level logic, and the in-game tutorial system delivered via the guide character.

TECHNICAL HIGHLIGHTS

  • Dual-challenge engine — quiz question timing and distraction spawning run on independent but coordinated systems
  • Distraction classification system — different types require different interaction patterns to resolve
  • Performance score model measuring accuracy, response time, and missed distractions
  • Progressive difficulty engine — increases distraction frequency, variety, and time pressure across levels
  • Post-game debrief linking player performance to real-world attention management strategies
